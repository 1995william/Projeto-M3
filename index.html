    A primeira reunião foi realizada através do zoom, para definir as tarefas, separar o que cada membro da equipe deveria fazer e também definir as datas para cada entrega.

    Trello

    O modelo de destribuição de tarefas utilizado foi o kanban, com a ferramente trello, que possibilitou a comunicação da equipe, e organização das atividades.!

    A o software utilizado para gerenciamento de dados foi o MySQL

    O GitHub foi utilizado para hospedagem e controle de versão dos dados.

10 Construtoras com mais vitórias em Grand Prêmios

A McLaren totalizou 179 vitórias, ficando atrás apenas da Ferrari que ficou em primeiro lugar com 240 vitórias.

  • Banco de dados: MySQL
  • Gráfico: Google Data Studio

Quantas corridas Lewis Hamilton venceu em cada construtora?

Hamilton venceu por 103 vezes, sendo 21 vitórias com McLaren e 82 vitórias com Mercedes!

  • Banco de dados: MySQL
  • Gráfico: Google Data Studio

5 pilotos mais jovens a vencer uma corrida na F1

Charles Leclerc e Max Verstappen são os dois pilotos mais jovens a vencer uma corrida na F1, ambos com 24 anos.

  • Banco de dados: MySQL
  • Gráfico: Google Data Studio

Quantas vezes um brasileiro venceu um Grande Prêmio?

Ayrton Senna é o brasileiro com mais vitórias em Grande Prêmio até o momento.

  • Banco de dados: MySQL
  • Gráfico: Google Data Studio

10 pilotos que mais conseguiram poles position na F1

Nas três primeiras posições temos o Hamilton em primeiro lugar com 103 viórias, em segundo lugar Michael Schumacher com 68, e em terceiro lugar Ayrton Senna com 60 viórias.

  • Banco de dados: MySQL
  • Gráfico: Google Data Studio

10 construtoras que mais ganharam Grand Prix tendo a pole position.

Em primeiro lugar temos a Ferrari representando a Itália com 100 vitórias. (As colunas indicam as marcas dos veículos, e as cores indicam os países.)

  • Banco de dados: MySQL
  • Gráfico: Google Data Studio

Quais as nacionalidades com maior quantidade de vitórias?

A Inglaterra é o país com maior número de construtoras campẽas na história da F1, com 494 vitórias.

  • Banco de dados: MySQL
  • Gráfico: Google Data Studio

10 pilotos que mais subiram no pódio (1ª 2ª e 3ª colocação)

Os três pilotos que mais subiram no podio na ordem 1ª 2ª e 3ª colocação foram, Lewis Hamilton, Michael Schumacher e Sebastian Vettel respectivamente.

  • Gráfico: Google Data Studio
